# Checkbox Component Skill

This skill helps you implement Material-UI (MUI v5) Checkbox components following the Minimals UI design system.

## Variants

### 1. Basic Checkbox
Simple checkbox implementation with default styling.

### 2. Sizes
Three size variations:
- Small (18px)
- Medium (20px) - default
- Large (24px)

### 3. Colors
Multiple color schemes:
- Primary (#00A76F - Minimals green)
- Secondary
- Error
- Warning
- Info
- Success
- Default

### 4. Indeterminate
Checkbox with indeterminate state (minus icon) for parent checkboxes in hierarchical lists.

### 5. With Label
Checkbox integrated with FormControlLabel for proper labeling and accessibility.

### 6. Controlled
Fully controlled checkbox with React state management.

### 7. Disabled
Non-interactive checkbox in disabled state (both checked and unchecked).

### 8. Custom Icon
Checkbox with custom icons for checked, unchecked, and indeterminate states.

### 9. Form Group
Multiple checkboxes grouped together using FormGroup for related options.

## When to Use

- Selection controls in forms
- Multi-select lists
- Terms and conditions acceptance
- Feature toggles
- Table row selection
- Filter panels

## Design System Compliance

All variants follow the Minimals UI design system with:
- Custom primary color (#00A76F)
- Consistent sizing (18px, 20px, 24px)
- 4px border-radius
- Proper spacing and alignment
- Ripple effects on interaction
- Accessible focus states
