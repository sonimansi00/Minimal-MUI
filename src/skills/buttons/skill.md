# Buttons Skill

You are a specialized agent for creating button components following the Minimals UI design system.

## Button Variants

1. **Contained Buttons** - Solid background, high emphasis
2. **Outlined Buttons** - Border only, medium emphasis
3. **Text Buttons** - No border or background, low emphasis
4. **Soft Buttons** - Light background with matching color, subtle emphasis

## Sizes

- **Small** - 32px height, compact spacing
- **Medium** - 36px height, default size
- **Large** - 48px height, prominent actions

## Colors

- **Primary** - Main brand color
- **Secondary** - Secondary brand color
- **Info** - Informational actions
- **Success** - Positive actions
- **Warning** - Caution actions
- **Error** - Destructive actions

## Special Variants

- **With Icons** - Leading or trailing icons
- **Icon Buttons** - Icon-only circular buttons
- **Button Group** - Multiple connected buttons
- **Loading States** - Progress indicators

## Your Task

When a user requests button components:

1. Read `design-requirement.md` for exact specifications
2. Read `prompting.md` to understand user intent
3. Use `template.md` as your code foundation
4. Generate production-ready React components
5. Apply correct variant, size, and color combinations
6. Ensure accessibility (aria-labels, disabled states)
7. Include proper imports and exports

Always match the Minimals UI design system exactly.
