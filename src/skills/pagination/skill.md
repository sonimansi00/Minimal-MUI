# Pagination Component Skill

Generate pagination components following Minimals design system patterns.

## Variants

### 1. Basic Pagination
- Default pagination with numbered pages
- Previous/Next navigation buttons
- Standard Material-UI styling

### 2. Outlined Pagination
- Outlined variant with borders
- Clean, minimalist appearance
- Better visibility on light backgrounds

### 3. Rounded Pagination
- Fully rounded button shapes
- Modern, pill-shaped design
- Smooth circular borders

### 4. Sizes
- **Small**: 32px height, compact spacing
- **Medium**: 40px height (default)
- **Large**: 48px height, prominent display

### 5. Colors
- **Primary**: Brand primary color
- **Secondary**: Secondary color scheme
- **Standard**: Default grey palette
- **Success**: Green success color
- **Error**: Red error color
- **Info**: Blue info color
- **Warning**: Orange warning color

### 6. Disabled State
- Greyed out appearance
- Non-interactive
- Reduced opacity

### 7. Controlled Pagination
- External state management
- onChange handler
- Current page tracking

### 8. Table Pagination
- Integrated with data tables
- Rows per page selector
- Page range display
- Navigation controls

### 9. Custom Icons
- Custom previous/next icons
- Icon customization
- Brand-specific navigation elements

## Component Features

- Responsive design
- Keyboard navigation support
- Accessibility compliant
- RTL support
- Customizable styling
- Type-safe props

## Usage Context

Use this skill when users request:
- Pagination controls
- Page navigation
- Table pagination
- Multi-page content navigation
- Data grid pagination
- List pagination
