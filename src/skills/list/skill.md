# List Component Skill

Generate Material-UI v5 List components following Minimals design system specifications.

## Available Variants

### 1. Basic List
- Simple list with text items
- Standard 56px item height
- Default padding and spacing

### 2. Dense List
- Compact list layout
- 40px item height
- Reduced padding for space efficiency

### 3. List with Avatar
- ListItemAvatar with circular avatar
- 56px item height
- Primary and secondary text support

### 4. List with Icon
- ListItemIcon with leading icon
- Icon alignment and spacing
- Text content with proper hierarchy

### 5. Nested List
- Expandable/collapsible sub-items
- Indentation for hierarchy
- Collapse/Expand icons
- Multiple nesting levels supported

### 6. Interactive List (Clickable)
- ListItemButton for clickable items
- Hover states with background color
- Ripple effects on click
- onClick handlers

### 7. Switch List
- Toggle switches aligned to the right
- Label text on the left
- Switch state management
- Secondary text optional

### 8. Checkbox List
- Checkboxes for multi-select
- Check/uncheck state handling
- Dense or standard variants
- Select all functionality optional

### 9. Selected Item List
- Single selection highlighting
- Selected state styling
- Background color change on selection
- Active item indication

## Design Specifications

- **Standard Item Height**: 56px
- **Dense Item Height**: 40px
- **Padding**: 16px horizontal, 8px vertical
- **Dividers**: 1px solid rgba(145, 158, 171, 0.2)
- **Hover State**: background rgba(145, 158, 171, 0.08)
- **Selected State**: background rgba(0, 184, 217, 0.08)
- **Icon Size**: 24px
- **Avatar Size**: 40px

## Usage

When a user requests a list component, analyze their requirements and generate the appropriate variant with proper MUI v5 components and Minimals styling.
